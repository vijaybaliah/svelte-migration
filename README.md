# Instructions

use node version `22.22.0`


## Step 1: Installation

Run the following commands

```sh
pnpm i
```

if you do not have pnpm use the instructions mentioned [here](https://pnpm.io/installation#using-other-package-managers)

## Step 2: Build

```sh
pnpm build
```

## Step 3:

```sh
cd apps/web
pnpm dev
```
